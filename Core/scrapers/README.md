## scrapers چی هستن اصلا ؟

هر scraper یک اسم داره. به طور مثال `line_1` این scraper اطلاعات فایل اکسل شرکت مترو تهران رو که مربوط به زمان بندی حرکت قطار های خط 1 (خط قرمز) میشه رو استخراج میکنه و در یک فایل `json`  که به نام همون خط هست به طور مثال برای خط 1 اسمش `stations_1.json` هست ذخیره میکنه. دخیره هم به صورت زیر خواهد بود :

```
{
	"Line name like Line_1" : {
		"Day of the week type like عادی یا پنجشنبه یا جمعه" : {
			"The station to which the train is moving like تئاترشهر" :{
				"stations name" : [
					"5:30",
					"5:40",
					...
				]
				
			}
		}
	}
}
```
## فایل run.py

این فایل کد اطلاعات فایل های اکسل خط ها رو استخراج میکنه و در فایل های `json` هم نام خط ها ذخیره میکنه. همچنین یک فایل `stations.json` هم میسازه که شامل نام تمام ایستگاه های خطوط هستش و به صورت زیر مرتب شده هستن :
```
{
	"stations" : {
		"Line name like Line_1" : [
			"stations name",
			...
		]
	}
}
```
