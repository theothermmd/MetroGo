### فایل های ثابت (و اینجور چیزها)

این فولدر شامل فایل های ثابت که از فایل stations_1.json تا stations_7.json شامل برنامه حرکت قطار های هر ایستگاه در روز های مختلف هفته هستش و فایل stations.json که شامل نام ایستگاه های هر خط هستش.

در فولدر  excels نیست فایل اسکل خط های مترو تهران از خط 1 تا خط 7 وجود داره.
