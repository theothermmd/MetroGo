سلام. تا حالا تو ایستگاه مترو گم شدید ؟ خب، متروگو اجازه نمیده این اتفاق بیفته! متروگو یک سرویس مسیریابی مترو ساده است که شما حتی نمیدونستید وجود داره.

این نسخه هسته یا بدون مخلفات API متروگو هستش. اگه متروگو وب، ماشین باشه، این انجین ماشین خواهد بود.


از گراف و الگوریتم dijkstra ( شاید A* بهتر باشه نه ؟)  برای پیدا کردن سریع ترین مسیر بین ایستگاه ها استفاده میکنه.

تا جایی که میتونه سریعه، پایتون است دیگر....

داستان جالبی برای سرویس دهی داره. اول از همه فایل app.py شامل کد های api هستش که با fastapi نوشته شده. و یک rate limiter هم برای جلوگیری از درخواست های زیاد داره. همچنین سعی بر این شده تو خود api مشکلات گرفته بشه مثل جلوگیری از اجرای جستجو برای سریع ترین مسیر در صورتی که به api ایستگاه مبدا یا مقصد ارسال نشده، تا اینجوری بار از روی سرور برداشته باشه.

در فولدر /Core چندین پوشه و یک فایل Core.py وجورد داره که میتونید اینجا داکیومنت هر کدوم رو بخوند.
