# متروگو چیه ؟

متروگو یک سرویس مسیریابی ساده ست برای مترو :) از یه سوال شروع شد. **" چطوری سریع ترین مسیر رو بین ایستگاه ها میشه پیدا کرد ؟! "**

# چه مشکلی رو حل میکنه ؟

متروگو بهتون کمک میکنه تا سریع ترین مسیر بین ایستگاه مبدا و مقصدتون رو پیدا کنید. همچنین میگه بهتون چند دقیقه دیگه قطار وارد ایستگاه مبدا میشه و به صورت دقیق میگه چه ساعتی به ایستگاه مقصد میرسید. همچنین در هنگام خط عوض کردن بهتون راهنمایی های لازم مثل کدوم خط سوار بشید و به سمت کدوم ایستگاه حرکت کنید رو میگه. و حتی یک راهنمای کوچیک داره برای کسانی که میخوان به صورت متنی بخونن چجوری میتونن از ایستگاه مبداشون برسن به ایستگاه مقصدشون :)
# چطوری کار میکنه ؟

متروگو از یک گراف برای ایستگاه ها و از الگوریتم Dijkstra ( شاید A* بهتر باشه نه ؟ ) برای پیدا کردن کمترین هزینه برای رفتن از ایستگاه مبدا به ایستگاه مقصد استفاده میکنه. زمان بندی حرکت قطار های مترو برای هر ایستگاه هم از فایل رسمی شرکت مترو که به صورت اکسل هستش دریافت میکنه و به مسیر اضافه میکنه. 

همچنین اگه علاقه مند به بیشتر دوستن راجب متروگو شدید پیشنهاد میکنم این داکیومنت رو مطالعه کنید :)

# چطوری در توسعه اش مشارکت کنم ؟

پروژه رو fork کنید. یه برانچ خفن بسازید مثل "{feature name} coolest branch in the world" و یکی از کارایی که در TODO پروژه هست رو بردارید و بنویسید و کامیت کنید و در آخر پول ریکوئست بزنید و ذوق کنیم و مرجش کنیم :)
